🚂 The RAPP Train

No dream deferred. Every person and every AI, working productively — together. The vehicle is RAPP.

One page, every ring of the RAPP Brainstem release train, runnable on any machine straight from here — no VPN, no signup.

Freshest → calmest: Canary → Nightly → Alpha → Beta → Grail. Rings are test builds qualified by CI + an attestation chain; only the Grail is the supported product. Test flights are sandboxed (~/.rapp-flight/, port 7075) and never touch an existing install. Joining makes a ring this machine's real install — switch rings by wiping ~/.brainstem first.

🏗️ The Shape Yard (deliveries into differently-shaped downstreams)

Rings test the payload; sidings test the deliveries. Nothing here touches grail, microsoft/aibast, or RAPP's lts main. Every distro's shape belongs to its distributor — and honoring that on every delivery is what lets anyone build on the kernel with confidence.
